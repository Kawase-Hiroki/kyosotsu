# kyosotsu

## Overview

This project simulates a dating app where useres interact under incomplete information depending on game theory. Each users have unique attractiveness index and disclose thier imformation to some extent. This simulator allows us to study convergence, efficiency and welfare properties.

## Motivation

This project is for use in a liberal arts final report class at institute of science tokyo. This project aims to obvious the reasons why many people feel dissatisfied with dating apps such as getting less match, popularity is oncentrated among a few.

## Usage

```bash
git clone https://github.com/Kawase-Hiroki/kyosotsu
cd kyosotsu
make
make plot
```
